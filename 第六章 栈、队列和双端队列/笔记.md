# 6.1 栈

**栈是由一系列对象组成的一个集合，这些对象的插入和删除操作遵循后进先出(LIFO)的原则。**

## 6.1.1 栈的抽象数据类型

**栈的操作：**
1. S.push(e)
2. S.pop(e)
3. S.top()
4. S.is_empty()
5. len(S)
